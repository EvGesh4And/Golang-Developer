# Channels in Go

[–°—Ç–∞—Ç—å—è](https://go101.org/article/channel.html)

–ö–∞–Ω–∞–ª—ã ‚Äî —ç—Ç–æ –æ–¥–∏–Ω –∏–∑ –∫–ª—é—á–µ–≤—ã—Ö –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã—Ö –º–µ—Ö–∞–Ω–∏–∑–º–æ–≤ –≤ Go, –¥–µ–ª–∞—é—â–∏–π –µ–≥–æ —É–Ω–∏–∫–∞–ª—å–Ω—ã–º. –í —Å–æ—á–µ—Ç–∞–Ω–∏–∏ —Å –≥–æ—Ä—É—Ç–∏–Ω–∞–º–∏ (goroutine) –∫–∞–Ω–∞–ª—ã (channel) –∑–Ω–∞—á–∏—Ç–µ–ª—å–Ω–æ —É–ø—Ä–æ—â–∞—é—Ç —Ä–∞–±–æ—Ç—É —Å –∫–æ–Ω–∫—É—Ä–µ–Ω—Ç–Ω—ã–º –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–µ–º (concurrent programming), –¥–µ–ª–∞—è –µ–≥–æ —É–¥–æ–±–Ω—ã–º –∏ –¥–∞–∂–µ —É–≤–ª–µ–∫–∞—Ç–µ–ª—å–Ω—ã–º.

–ö–∞–Ω–∞–ª –≤ –æ—Å–Ω–æ–≤–Ω–æ–º –≤—ã—Å—Ç—É–ø–∞–µ—Ç –≤ –∫–∞—á–µ—Å—Ç–≤–µ –º–µ—Ç–æ–¥–∞ [—Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏–∏ –∫–æ–Ω–∫—É—Ä–µ–Ω—Ç–Ω–æ—Å—Ç–∏](https://go101.org/article/control-flows-more.html#synchronization). –í —ç—Ç–æ–π —Å—Ç–∞—Ç—å–µ –±—É–¥—É—Ç –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω—ã –≤—Å–µ –∫–æ–Ω—Ü–µ–ø—Ü–∏–∏, —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –∏ –ø—Ä–∞–≤–∏–ª–∞, —Å–≤—è–∑–∞–Ω–Ω—ã–µ —Å –∫–∞–Ω–∞–ª–æ–º. –ß—Ç–æ–±—ã –ª—É—á—à–µ –ø–æ–Ω—è—Ç—å –∫–∞–Ω–∞–ª—ã, —Ç–∞–∫–∂–µ –ø—Ä–æ—Å—Ç–æ –æ–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è –≤–Ω—É—Ç—Ä–µ–Ω–Ω—è—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –∫–∞–Ω–∞–ª–æ–≤ –∏ –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –¥–µ—Ç–∞–ª–∏ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–º –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–æ–º/—Å—Ä–µ–¥–æ–π –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è Go.

–ò–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è –≤ —ç—Ç–æ–π —Å—Ç–∞—Ç—å–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–µ–º–Ω–æ–≥–æ —Å–ª–æ–∂–Ω–æ–π –¥–ª—è –Ω–æ–≤—ã—Ö —Å—É—Å–ª–∏–∫–æ–≤. –ù–µ–∫–æ—Ç–æ—Ä—ã–µ —á–∞—Å—Ç–∏ —ç—Ç–æ–π —Å—Ç–∞—Ç—å–∏, –≤–æ–∑–º–æ–∂–Ω–æ, –ø—Ä–∏–¥–µ—Ç—Å—è –ø—Ä–æ—á–∏—Ç–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ä–∞–∑, —á—Ç–æ–±—ã –ø–æ–ª–Ω–æ—Å—Ç—å—é –ø–æ–Ω—è—Ç—å.

## –í–≤–µ–¥–µ–Ω–∏–µ –≤ –∫–∞–Ω–∞–ª—ã (Channel Introduction)

–û–¥–Ω–æ –∏–∑ –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–∏–π (—Å–¥–µ–ª–∞–Ω–Ω–æ–µ *–†–æ–±–æ–º –ü–∞–π–∫–æ–º*) –ø–æ –∫–æ–Ω–∫—É—Ä–µ–Ω—Ç–Ω–æ–º—É –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—é –∑–≤—É—á–∏—Ç —Ç–∞–∫:
> **"–ù–µ (–ø–æ–∑–≤–æ–ª—è–π—Ç–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è–º) –æ–±–º–µ–Ω–∏–≤–∞—Ç—å—Å—è –¥–∞–Ω–Ω—ã–º–∏, —Å–æ–≤–º–µ—Å—Ç–Ω–æ –∏—Å–ø–æ–ª—å–∑—É—è –ø–∞–º—è—Ç—å, (–ø–æ–∑–≤–æ–ª—å—Ç–µ –∏–º) —Å–æ–≤–º–µ—Å—Ç–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –ø–∞–º—è—Ç—å, –æ–±–º–µ–Ω–∏–≤–∞—è—Å—å –¥–∞–Ω–Ω—ã–º–∏ (–ø–æ –∫–∞–Ω–∞–ª–∞–º)."**  
> *("Don't (let computations) communicate by sharing memory, (let them) share memory by communicating (through channels)")*

(–í –∫–æ–Ω—Ç–µ–∫—Å—Ç–µ Go –∫–∞–∂–¥–æ–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ –º–æ–∂–Ω–æ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å –∫–∞–∫ –≥–æ—Ä—É—Ç–∏–Ω—É.)

**–ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –æ–±—â–µ–π –ø–∞–º—è—Ç–∏ –¥–ª—è –æ–±–º–µ–Ω–∞ –¥–∞–Ω–Ω—ã–º–∏** –∏ **–æ–±–º–µ–Ω –¥–∞–Ω–Ω—ã–º–∏ –¥–ª—è —Å–æ–≤–º–µ—Å—Ç–Ω–æ–≥–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –ø–∞–º—è—Ç–∏** ‚Äî —ç—Ç–æ –¥–≤–∞ —Ä–∞–∑–Ω—ã—Ö –ø–æ–¥—Ö–æ–¥–∞ –∫ –∫–æ–Ω–∫—É—Ä–µ–Ω—Ç–Ω–æ–º—É –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—é. –ö–æ–≥–¥–∞ –≥–æ—Ä—É—Ç–∏–Ω—ã –æ–±–º–µ–Ω–∏–≤–∞—é—Ç—Å—è –¥–∞–Ω–Ω—ã–º–∏ —á–µ—Ä–µ–∑ –æ–±—â—É—é –ø–∞–º—è—Ç—å, –ø—Ä–∏–º–µ–Ω—è—é—Ç—Å—è —Ç—Ä–∞–¥–∏—Ü–∏–æ–Ω–Ω—ã–µ –º–µ—Ç–æ–¥—ã —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏–∏, —Ç–∞–∫–∏–µ –∫–∞–∫ –º—å—é—Ç–µ–∫—Å—ã, –¥–ª—è –∑–∞—â–∏—Ç—ã –æ–±—â–µ–π –ø–∞–º—è—Ç–∏ –∏ –ø—Ä–µ–¥–æ—Ç–≤—Ä–∞—â–µ–Ω–∏—è –≥–æ–Ω–∫–∏ –¥–∞–Ω–Ω—ã—Ö. –û–¥–Ω–∞–∫–æ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–∞–Ω–∞–ª—ã, —á—Ç–æ–±—ã —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –≤—Ç–æ—Ä–æ–π –ø–æ–¥—Ö–æ–¥ ‚Äî –æ–±–º–µ–Ω –¥–∞–Ω–Ω—ã–º–∏ –¥–ª—è —Å–æ–≤–º–µ—Å—Ç–Ω–æ–≥–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –ø–∞–º—è—Ç–∏.

Go –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è–µ—Ç —É–Ω–∏–∫–∞–ª—å–Ω—ã–π –º–µ—Ö–∞–Ω–∏–∑–º —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏–∏ ‚Äî **–∫–∞–Ω–∞–ª—ã (channel)**. –ö–∞–Ω–∞–ª—ã –ø–æ–∑–≤–æ–ª—è—é—Ç –≥–æ—Ä—É—Ç–∏–Ω–∞–º —Å–æ–≤–º–µ—Å—Ç–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –ø–∞–º—è—Ç—å —á–µ—Ä–µ–∑ –æ–±–º–µ–Ω –¥–∞–Ω–Ω—ã–º–∏. –ö–∞–Ω–∞–ª –º–æ–∂–Ω–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç—å –∫–∞–∫ –≤–Ω—É—Ç—Ä–µ–Ω–Ω—é—é –æ—á–µ—Ä–µ–¥—å **FIFO (–ø–µ—Ä–≤—ã–º –ø—Ä–∏—à–µ–ª ‚Äî –ø–µ—Ä–≤—ã–º –≤—ã—à–µ–ª)** –≤–Ω—É—Ç—Ä–∏ –ø—Ä–æ–≥—Ä–∞–º–º—ã. –û–¥–Ω–∏ –≥–æ—Ä—É—Ç–∏–Ω—ã –æ—Ç–ø—Ä–∞–≤–ª—è—é—Ç –≤ –∫–∞–Ω–∞–ª –∑–Ω–∞—á–µ–Ω–∏—è, –∞ –¥—Ä—É–≥–∏–µ –ø–æ–ª—É—á–∞—é—Ç –∏—Ö.

–ü–æ–º–∏–º–æ –ø–µ—Ä–µ–¥–∞—á–∏ —Å–∞–º–∏—Ö –∑–Ω–∞—á–µ–Ω–∏–π —á–µ—Ä–µ–∑ –∫–∞–Ω–∞–ª—ã, –º–µ–∂–¥—É –≥–æ—Ä—É—Ç–∏–Ω–∞–º–∏ –º–æ–∂–µ—Ç –ø–µ—Ä–µ–¥–∞–≤–∞—Ç—å—Å—è –∏ –≤–ª–∞–¥–µ–Ω–∏–µ —ç—Ç–∏–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏. –ö–æ–≥–¥–∞ –≥–æ—Ä—É—Ç–∏–Ω–∞ –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ –≤ –∫–∞–Ω–∞–ª, –º–æ–∂–Ω–æ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å —ç—Ç–æ –∫–∞–∫ –æ—Å–≤–æ–±–æ–∂–¥–µ–Ω–∏–µ –µ—é –≤–ª–∞–¥–µ–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏–µ–º (–∫–æ—Ç–æ—Ä–æ–µ –º–æ–≥–ª–æ –±—ã—Ç—å –¥–æ—Å—Ç—É–ø–Ω–æ —á–µ—Ä–µ–∑ –ø–µ—Ä–µ–¥–∞–Ω–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ). –°–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ, –∫–æ–≥–¥–∞ –¥—Ä—É–≥–∞—è –≥–æ—Ä—É—Ç–∏–Ω–∞ –ø–æ–ª—É—á–∞–µ—Ç —ç—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ –∏–∑ –∫–∞–Ω–∞–ª–∞, –º–æ–∂–Ω–æ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å —ç—Ç–æ –∫–∞–∫ –ø—Ä–∏–æ–±—Ä–µ—Ç–µ–Ω–∏–µ –≤–ª–∞–¥–µ–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏–µ–º.

–†–∞–∑—É–º–µ–µ—Ç—Å—è, –Ω–µ –≤–æ –≤—Å–µ—Ö —Å–ª—É—á–∞—è—Ö –ø–µ—Ä–µ–¥–∞—á–∞ –¥–∞–Ω–Ω—ã—Ö —á–µ—Ä–µ–∑ –∫–∞–Ω–∞–ª –æ–∑–Ω–∞—á–∞–µ—Ç –ø–µ—Ä–µ–¥–∞—á—É –≤–ª–∞–¥–µ–Ω–∏—è. –ó–Ω–∞—á–µ–Ω–∏—è, –≤–ª–∞–¥–µ–Ω–∏–µ –∫–æ—Ç–æ—Ä—ã–º–∏ –ø–µ—Ä–µ–¥–∞–µ—Ç—Å—è, —á–∞—Å—Ç–æ (–Ω–æ –Ω–µ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ) –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç —Å–æ–±–æ–π —Å—Å—ã–ª–∫–∏. –í–∞–∂–Ω–æ –ø–æ–Ω–∏–º–∞—Ç—å, —á—Ç–æ —Ä–µ—á—å –∏–¥–µ—Ç –æ –ª–æ–≥–∏—á–µ—Å–∫–æ–º –≤–ª–∞–¥–µ–Ω–∏–∏. –ö–∞–Ω–∞–ª—ã –≤ Go –ø–æ–º–æ–≥–∞—é—Ç –ø–∏—Å–∞—Ç—å –∫–æ–¥ –±–µ–∑ –≥–æ–Ω–∫–∏ –¥–∞–Ω–Ω—ã—Ö, –Ω–æ –Ω–µ –ø—Ä–µ–¥–æ—Ç–≤—Ä–∞—â–∞—é—Ç –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –Ω–∞–ø–∏—Å–∞–Ω–∏—è –Ω–µ—ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ–≥–æ –∏–ª–∏ –Ω–µ–∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ–≥–æ –∫–æ–Ω–∫—É—Ä–µ–Ω—Ç–Ω–æ–≥–æ –∫–æ–¥–∞.

**–•–æ—Ç—è Go –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –∏ —Ç—Ä–∞–¥–∏—Ü–∏–æ–Ω–Ω—ã–µ –º–µ—Ç–æ–¥—ã —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏–∏, —Ç–æ–ª—å–∫–æ –∫–∞–Ω–∞–ª—ã —è–≤–ª—è—é—Ç—Å—è –ø–æ–ª–Ω–æ—Ü–µ–Ω–Ω—ã–º —ç–ª–µ–º–µ–Ω—Ç–æ–º —è–∑—ã–∫–∞.** –ö–∞–Ω–∞–ª—ã ‚Äî —ç—Ç–æ –æ—Ç–¥–µ–ª—å–Ω—ã–π —Ç–∏–ø –¥–∞–Ω–Ω—ã—Ö –≤ Go, –ø–æ—ç—Ç–æ–º—É –∏—Ö –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –±–µ–∑ –ø–æ–¥–∫–ª—é—á–µ–Ω–∏—è –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã—Ö –ø–∞–∫–µ—Ç–æ–≤. –í —Ç–æ –∂–µ –≤—Ä–µ–º—è —Ç—Ä–∞–¥–∏—Ü–∏–æ–Ω–Ω—ã–µ –º–µ—Ö–∞–Ω–∏–∑–º—ã —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏–∏ —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω—ã –≤ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã—Ö –ø–∞–∫–µ—Ç–∞—Ö `sync` –∏ `sync/atomic`.

–ö–∞–∂–¥—ã–π –º–µ—Ö–∞–Ω–∏–∑–º —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏–∏ –ª—É—á—à–µ –≤—Å–µ–≥–æ –ø–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã—Ö —Å—Ü–µ–Ω–∞—Ä–∏–µ–≤. –û–¥–Ω–∞–∫–æ –∫–∞–Ω–∞–ª—ã –æ–±–ª–∞–¥–∞—é—Ç —à–∏—Ä–æ–∫–∏–º–∏ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—è–º–∏ –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è –∏ –±–æ–ª—å—à–∏–º —Ä–∞–∑–Ω–æ–æ–±—Ä–∞–∑–∏–µ–º —Å–ø–æ—Å–æ–±–æ–≤ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è. –û–¥–Ω–∞ –∏–∑ –ø—Ä–æ–±–ª–µ–º –∫–∞–Ω–∞–ª–æ–≤ –≤ —Ç–æ–º, —á—Ç–æ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–µ —Å –∏—Ö –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –Ω–∞—Å—Ç–æ–ª—å–∫–æ —É–¥–æ–±–Ω–æ–µ –∏ –ø—Ä–∏—è—Ç–Ω–æ–µ, —á—Ç–æ —Ä–∞–∑—Ä–∞–±–æ—Ç—á–∏–∫–∏ –º–æ–≥—É—Ç –ø—Ä–∏–º–µ–Ω—è—Ç—å –∏—Ö –¥–∞–∂–µ —Ç–∞–º, –≥–¥–µ –æ–Ω–∏ –Ω–µ —è–≤–ª—è—é—Ç—Å—è –Ω–∞–∏–ª—É—á—à–∏–º —Ä–µ—à–µ–Ω–∏–µ–º.

## –¢–∏–ø—ã –∏ –∑–Ω–∞—á–µ–Ω–∏—è –∫–∞–Ω–∞–ª–æ–≤

–ö–∞–∫ –∏ –º–∞—Å—Å–∏–≤—ã, —Å—Ä–µ–∑—ã –∏ –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏—è (map), –∫–∞–∂–¥—ã–π —Ç–∏–ø –∫–∞–Ω–∞–ª–∞ –∏–º–µ–µ—Ç —Ç–∏–ø —ç–ª–µ–º–µ–Ω—Ç–æ–≤. –ö–∞–Ω–∞–ª –º–æ–∂–µ—Ç –ø–µ—Ä–µ–¥–∞–≤–∞—Ç—å —Ç–æ–ª—å–∫–æ –∑–Ω–∞—á–µ–Ω–∏—è —ç—Ç–æ–≥–æ —Ç–∏–ø–∞.

–ö–∞–Ω–∞–ª—ã –±—ã–≤–∞—é—Ç –¥–≤—É–Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–Ω—ã–º–∏ –∏ –æ–¥–Ω–æ–Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–Ω—ã–º–∏. –ü—É—Å—Ç—å `T` ‚Äî –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω—ã–π —Ç–∏–ø:

1. `chan T` –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç –¥–≤—É–Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–Ω—ã–π –∫–∞–Ω–∞–ª. –ö–æ–º–ø–∏–ª—è—Ç–æ—Ä —Ä–∞–∑—Ä–µ—à–∞–µ—Ç –∏ –æ—Ç–ø—Ä–∞–≤–∫—É, –∏ –ø–æ–ª—É—á–µ–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏–π –∏–∑ —Ç–∞–∫–∏—Ö –∫–∞–Ω–∞–ª–æ–≤.
2. `chan<- T` –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç –∫–∞–Ω–∞–ª —Ç–æ–ª—å–∫–æ –¥–ª—è –æ—Ç–ø—Ä–∞–≤–∫–∏ (send-only). –ö–æ–º–ø–∏–ª—è—Ç–æ—Ä –Ω–µ –ø–æ–∑–≤–æ–ª—è–µ—Ç –ø–æ–ª—É—á–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è –∏–∑ —Ç–∞–∫–∏—Ö –∫–∞–Ω–∞–ª–æ–≤.
3. `<-chan T` –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç –∫–∞–Ω–∞–ª —Ç–æ–ª—å–∫–æ –¥–ª—è –ø–æ–ª—É—á–µ–Ω–∏—è (receive-only). –ö–æ–º–ø–∏–ª—è—Ç–æ—Ä –Ω–µ –ø–æ–∑–≤–æ–ª—è–µ—Ç –æ—Ç–ø—Ä–∞–≤–ª—è—Ç—å –≤ –Ω–µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è.

`T` –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —Ç–∏–ø–æ–º —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —ç—Ç–∏—Ö –∫–∞–Ω–∞–ª–æ–≤.

–ó–Ω–∞—á–µ–Ω–∏—è –¥–≤—É–Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–Ω–æ–≥–æ —Ç–∏–ø–∞ `chan T` –º–æ–≥—É—Ç –Ω–µ—è–≤–Ω–æ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤—ã–≤–∞—Ç—å—Å—è –∫–∞–∫ –≤ `chan<- T`, —Ç–∞–∫ –∏ –≤ `<-chan T`, –Ω–æ –æ–±—Ä–∞—Ç–Ω–æ–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –Ω–µ–≤–æ–∑–º–æ–∂–Ω–æ, –¥–∞–∂–µ —è–≤–Ω–æ. –ó–Ω–∞—á–µ–Ω–∏—è —Ç–∏–ø–∞ `chan<- T` –Ω–µ–ª—å–∑—è –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞—Ç—å –≤ `<-chan T` –∏ –Ω–∞–æ–±–æ—Ä–æ—Ç.
–°–ª–µ–¥—É–µ—Ç –æ—Ç–º–µ—Ç–∏—Ç—å, —á—Ç–æ `<-` –≤ –æ–±—ä—è–≤–ª–µ–Ω–∏—è—Ö —Ç–∏–ø–æ–≤ –∫–∞–Ω–∞–ª–æ–≤ —è–≤–ª—è–µ—Ç—Å—è –º–æ–¥–∏—Ñ–∏–∫–∞—Ç–æ—Ä–æ–º, –∞ –Ω–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–º.

–ö–∞–∂–¥–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∫–∞–Ω–∞–ª–∞ –∏–º–µ–µ—Ç –µ–º–∫–æ—Å—Ç—å (capacity), –∫–æ—Ç–æ—Ä–∞—è –±—É–¥–µ—Ç —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–Ω–∞ –≤ —Å–ª–µ–¥—É—é—â–µ–º —Ä–∞–∑–¥–µ–ª–µ.

* –ö–∞–Ω–∞–ª —Å –µ–º–∫–æ—Å—Ç—å—é 0 –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –Ω–µ–±—É—Ñ–µ—Ä–∏–∑–æ–≤–∞–Ω–Ω—ã–º (unbuffered).
* –ö–∞–Ω–∞–ª —Å –µ–º–∫–æ—Å—Ç—å—é –±–æ–ª—å—à–µ 0 –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –±—É—Ñ–µ—Ä–∏–∑–æ–≤–∞–Ω–Ω—ã–º (buffered).

–ó–Ω–∞—á–µ–Ω–∏—è –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é (zero values) –¥–ª—è —Ç–∏–ø–æ–≤ –∫–∞–Ω–∞–ª–æ–≤ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ã –ø—Ä–µ–¥–æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–º –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä–æ–º nil.
–ß—Ç–æ–±—ã —Å–æ–∑–¥–∞—Ç—å –Ω–µ nil –∫–∞–Ω–∞–ª, –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –≤—Å—Ç—Ä–æ–µ–Ω–Ω—É—é —Ñ—É–Ω–∫—Ü–∏—é make.

–ü—Ä–∏–º–µ—Ä:
```go
make(chan int, 10)
```

–≠—Ç–æ—Ç –≤—ã–∑–æ–≤ —Å–æ–∑–¥–∞—Å—Ç –∫–∞–Ω–∞–ª —Å —Ç–∏–ø–æ–º —ç–ª–µ–º–µ–Ω—Ç–æ–≤ int –∏ –µ–º–∫–æ—Å—Ç—å—é 10.

–í—Ç–æ—Ä–æ–π –∞—Ä–≥—É–º–µ–Ω—Ç –≤ make –∑–∞–¥–∞–µ—Ç –µ–º–∫–æ—Å—Ç—å –∫–∞–Ω–∞–ª–∞. –û–Ω –Ω–µ–æ–±—è–∑–∞—Ç–µ–ª—å–Ω—ã–π; –µ—Å–ª–∏ –µ–≥–æ –Ω–µ —É–∫–∞–∑–∞—Ç—å, –µ–º–∫–æ—Å—Ç—å –±—É–¥–µ—Ç —Ä–∞–≤–Ω–∞ 0.

## –°—Ä–∞–≤–Ω–µ–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏–π –∫–∞–Ω–∞–ª–æ–≤

–í—Å–µ —Ç–∏–ø—ã –∫–∞–Ω–∞–ª–æ–≤ –≤ Go —è–≤–ª—è—é—Ç—Å—è —Å—Ä–∞–≤–Ω–∏–º—ã–º–∏ (comparable).

–ò–∑ —Å—Ç–∞—Ç—å–∏ [value parts](https://go101.org/article/value-part.html) –º—ã –∑–Ω–∞–µ–º, —á—Ç–æ –∑–Ω–∞—á–µ–Ω–∏—è –∫–∞–Ω–∞–ª–∞, –æ—Ç–ª–∏—á–Ω—ã–µ –æ—Ç nil, —è–≤–ª—è—é—Ç—Å—è –∑–Ω–∞—á–µ–Ω–∏—è–º–∏, —Å–æ—Å—Ç–æ—è—â–∏–º–∏ –∏–∑ –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö —á–∞—Å—Ç–µ–π. –ï—Å–ª–∏ –æ–¥–Ω–æ –∑–Ω–∞—á–µ–Ω–∏–µ –∫–∞–Ω–∞–ª–∞ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è –¥—Ä—É–≥–æ–º—É, —Ç–æ –æ–±–∞ –∫–∞–Ω–∞–ª–∞ –∏—Å–ø–æ–ª—å–∑—É—é—Ç –æ–¥–Ω—É –∏ —Ç—É –∂–µ –±–∞–∑–æ–≤—É—é —á–∞—Å—Ç—å. –î—Ä—É–≥–∏–º–∏ —Å–ª–æ–≤–∞–º–∏, —ç—Ç–∏ –¥–≤–∞ –∫–∞–Ω–∞–ª–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç –æ–¥–∏–Ω –∏ —Ç–æ—Ç –∂–µ –æ–±—ä–µ–∫—Ç –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–≥–æ –∫–∞–Ω–∞–ª–∞. –†–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –∏—Ö —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —è–≤–ª—è–µ—Ç—Å—è `true`.

## –û–ø–µ—Ä–∞—Ü–∏–∏ —Å –∫–∞–Ω–∞–ª–∞–º–∏
–í Go –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–æ –ø—è—Ç—å –æ–ø–µ—Ä–∞—Ü–∏–π –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å –∫–∞–Ω–∞–ª–∞–º–∏. –ü—É—Å—Ç—å `ch` ‚Äî –Ω–µ–∫–æ—Ç–æ—Ä—ã–π –∫–∞–Ω–∞–ª, —Ç–æ–≥–¥–∞:

1. –ó–∞–∫—Ä–æ–π—Ç–µ –∫–∞–Ω–∞–ª, –∏—Å–ø–æ–ª—å–∑—É—è —Å–ª–µ–¥—É—é—â–∏–π –≤—ã–∑–æ–≤ —Ñ—É–Ω–∫—Ü–∏–∏
	```go
	close(ch)
	```
	–≥–¥–µ `close` - —ç—Ç–æ –≤—Å—Ç—Ä–æ–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è. –ê—Ä–≥—É–º–µ–Ω—Ç–æ–º –≤—ã–∑–æ–≤–∞ —Ñ—É–Ω–∫—Ü–∏–∏ `close` –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –∫–∞–Ω–∞–ª–∞, –∞ –∫–∞–Ω–∞–ª `ch` –Ω–µ –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –∫–∞–Ω–∞–ª–æ–º —Ç–æ–ª—å–∫–æ –¥–ª—è –ø—Ä–∏–µ–º–∞ (`<-chan T`).

2. –û—Ç–ø—Ä–∞–≤—å—Ç–µ –∑–Ω–∞—á–µ–Ω–∏–µ `v` –≤ –∫–∞–Ω–∞–ª, –∏—Å–ø–æ–ª—å–∑—É—è —Å–ª–µ–¥—É—é—â–∏–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å
	```go
	ch <- v
	```
	–≥–¥–µ `v` –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ–º, –∫–æ—Ç–æ—Ä–æ–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –ø—Ä–∏—Å–≤–æ–µ–Ω–æ —Ç–∏–ø—É —ç–ª–µ–º–µ–Ω—Ç–∞ –∫–∞–Ω–∞–ª–∞ `ch`, –∏ –∫–∞–Ω–∞–ª `ch` –Ω–µ –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –∫–∞–Ω–∞–ª–æ–º —Ç–æ–ª—å–∫–æ –¥–ª—è –ø—Ä–∏–µ–º–∞ (`<-chan T`). –û–ø–µ—Ä–∞—Ç–æ—Ä `<-` –∑–¥–µ—Å—å ‚Äî **–æ–ø–µ—Ä–∞—Ç–æ—Ä –æ—Ç–ø—Ä–∞–≤–∫–∏**.

3. –ü–æ–ª—É—á–µ–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è –∏–∑ –∫–∞–Ω–∞–ª–∞ —Å –ø–æ–º–æ—â—å—é —Å–ª–µ–¥—É—é—â–µ–≥–æ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å–∞

	```go
	<-ch
	```

	–û–ø–µ—Ä–∞—Ü–∏—è –ø–æ–ª—É—á–µ–Ω–∏—è –ø–æ –∫–∞–Ω–∞–ª—É –≤—Å–µ–≥–¥–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø–æ –∫—Ä–∞–π–Ω–µ–π –º–µ—Ä–µ –æ–¥–∏–Ω —Ä–µ–∑—É–ª—å—Ç–∞—Ç, –∫–æ—Ç–æ—Ä—ã–π —è–≤–ª—è–µ—Ç—Å—è –∑–Ω–∞—á–µ–Ω–∏–µ–º —Ç–∏–ø–∞ —ç–ª–µ–º–µ–Ω—Ç–∞ –∫–∞–Ω–∞–ª–∞, –∏ –∫–∞–Ω–∞–ª `ch` –Ω–µ –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –∫–∞–Ω–∞–ª–æ–º —Ç–æ–ª—å–∫–æ –¥–ª—è –æ—Ç–ø—Ä–∞–≤–∫–∏ (`chan<- T`). –û–ø–µ—Ä–∞—Ç–æ—Ä `<-` –≤ —ç—Ç–æ–º —Å–ª—É—á–∞–µ ‚Äî **–æ–ø–µ—Ä–∞—Ç–æ—Ä –ø–æ–ª—É—á–µ–Ω–∏—è**.

	–í –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–µ —Å–ª—É—á–∞–µ–≤ –ø–æ–ª—É—á–µ–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è –∏–∑ –∫–∞–Ω–∞–ª–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –∫–∞–∫ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ —Å –æ–¥–Ω–∏–º –∑–Ω–∞—á–µ–Ω–∏–µ–º:
	```go
	v = <-ch
	```
	–û–¥–Ω–∞–∫–æ, –∫–æ–≥–¥–∞ –æ–ø–µ—Ä–∞—Ü–∏—è —Å –∫–∞–Ω–∞–ª–æ–º –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤ –∫–∞—á–µ—Å—Ç–≤–µ –µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω–æ–≥–æ –∏—Å—Ö–æ–¥–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è –≤ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–∏, —Ä–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –º–æ–∂–µ—Ç –±—ã—Ç—å –≤—Ç–æ—Ä–æ–µ –Ω–µ–æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ–µ –Ω–µ—Ç–∏–ø–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –ª–æ–≥–∏—á–µ—Å–∫–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∏ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –º–Ω–æ–≥–æ–∑–Ω–∞—á–Ω—ã–º.
	
	–û–¥–Ω–∞–∫–æ, –µ—Å–ª–∏ –∫–∞–Ω–∞–ª –∑–∞–∫—Ä—ã—Ç, –º–æ–∂–Ω–æ –ø—Ä–æ–≤–µ—Ä–∏—Ç—å, –±—ã–ª–æ –ª–∏ –∑–Ω–∞—á–µ–Ω–∏–µ –æ—Ç–ø—Ä–∞–≤–ª–µ–Ω–æ **–¥–æ –∑–∞–∫—Ä—ã—Ç–∏—è**:
	```go
	v, sentBeforeClosed = <-ch
	```
	–ó–¥–µ—Å—å `sentBeforeClosed` ‚Äî –±—É–ª–µ–≤–æ –∑–Ω–∞—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ —Ä–∞–≤–Ω–æ `false`, –µ—Å–ª–∏ –∫–∞–Ω–∞–ª –±—ã–ª –∑–∞–∫—Ä—ã—Ç –∏ –±–æ–ª—å—à–µ –≤ –Ω–µ–º –Ω–µ—Ç –¥–∞–Ω–Ω—ã—Ö.
	(–ù–∏–∂–µ –º—ã —É–∑–Ω–∞–µ–º, —á—Ç–æ –º—ã –º–æ–∂–µ–º –ø–æ–ª—É—á–∞—Ç—å –Ω–µ–æ–≥—Ä–∞–Ω–∏—á–µ–Ω–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∑–Ω–∞—á–µ–Ω–∏–π –∏–∑ –∑–∞–∫—Ä—ã—Ç–æ–≥–æ –∫–∞–Ω–∞–ª–∞.)

4. –ü–æ–ª—É—á–µ–Ω–∏–µ –µ–º–∫–æ—Å—Ç–∏ –∫–∞–Ω–∞–ª–∞
	```go
	cap(ch)
	```
	–≥–¥–µ `cap` - —ç—Ç–æ –≤—Å—Ç—Ä–æ–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è –∫–æ–≥–¥–∞-–ª–∏–±–æ –±—ã–ª–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∞ –≤ [–∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞—Ö –≤ Go](https://go101.org/article/container.html#cap-len). –†–µ–∑—É–ª—å—Ç–∞—Ç–æ–º –≤—ã–∑–æ–≤–∞ —Ñ—É–Ω–∫—Ü–∏–∏ `cap` —è–≤–ª—è–µ—Ç—Å—è –∑–Ω–∞—á–µ–Ω–∏–µ `int`.
5. –ü–æ–ª—É—á–µ–Ω–∏–µ —Ç–µ–∫—É—â–µ–π –¥–ª–∏–Ω—ã –∫–∞–Ω–∞–ª–∞
   	```go
   	len(ch)
	```
	–≥–¥–µ `len` - —ç—Ç–æ –≤—Å—Ç—Ä–æ–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è —Ç–∞–∫–∂–µ –∫–æ–≥–¥–∞-–ª–∏–±–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª–∞—Å—å —Ä–∞–Ω–µ–µ. –í–æ–∑–≤—Ä–∞—â–∞–µ–º–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –ø—Ä–∏ –≤—ã–∑–æ–≤–µ —Ñ—É–Ω–∫—Ü–∏–∏ `len` - —ç—Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ —Ç–∏–ø–∞ `int`. –†–µ–∑—É–ª—å—Ç–∏—Ä—É—é—â–∞—è –¥–ª–∏–Ω–∞ - —ç—Ç–æ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ —É–∂–µ –±—ã–ª–∏ —É—Å–ø–µ—à–Ω–æ –æ—Ç–ø—Ä–∞–≤–ª–µ–Ω—ã –≤ –∑–∞–ø—Ä–æ—à–µ–Ω–Ω—ã–π –∫–∞–Ω–∞–ª, –Ω–æ –µ—â–µ –Ω–µ –±—ã–ª–∏ –ø–æ–ª—É—á–µ–Ω—ã (–∏–∑–≤–ª–µ—á–µ–Ω—ã).

–ë–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ –±–∞–∑–æ–≤—ã—Ö –æ–ø–µ—Ä–∞—Ü–∏–π –≤ Go –Ω–µ —è–≤–ª—è—é—Ç—Å—è —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω—ã–º–∏. –î—Ä—É–≥–∏–º–∏ —Å–ª–æ–≤–∞–º–∏, –æ–Ω–∏ –Ω–µ —è–≤–ª—è—é—Ç—Å—è –ø–æ—Ç–æ–∫–æ–±–µ–∑–æ–ø–∞—Å–Ω—ã–º–∏ ( –æ–Ω–∏ –Ω–µ —è–≤–ª—è—é—Ç—Å—è –±–µ–∑–æ–ø–∞—Å–Ω—ã–º–∏ —Å —Ç–æ—á–∫–∏ –∑—Ä–µ–Ω–∏—è –∫–æ–Ω–∫—É—Ä–µ–Ω—Ç–Ω–æ—Å—Ç–∏). –ö —Ç–∞–∫–∏–º –æ–ø–µ—Ä–∞—Ü–∏—è–º –æ—Ç–Ω–æ—Å—è—Ç—Å—è
- –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏–π,
- –ø–µ—Ä–µ–¥–∞—á–∞ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤,
- –º–∞–Ω–∏–ø—É–ª—è—Ü–∏–∏ —Å —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–æ–≤ –∏ —Ç. –¥.
–û–¥–Ω–∞–∫–æ –≤—Å–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å –∫–∞–Ω–∞–ª–∞–º–∏ —É–∂–µ —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∏—Ä–æ–≤–∞–Ω—ã, –ø–æ—ç—Ç–æ–º—É –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–∞—è —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏—è –Ω–µ —Ç—Ä–µ–±—É–µ—Ç—Å—è –¥–ª—è –±–µ–∑–æ–ø–∞—Å–Ω–æ–≥–æ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è —ç—Ç–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π.

–ï—Å–ª–∏ –ø—Ä–æ–≤–µ—Ä—è–µ–º—ã–π –∫–∞–Ω–∞–ª —è–≤–ª—è–µ—Ç—Å—è `nil`-–∫–∞–Ω–∞–ª–æ–º, —Ç–æ –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ `cap` –∏ `len` –≤–µ—Ä–Ω—É—Ç –Ω–æ–ª—å. –≠—Ç–∏ –¥–≤–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –Ω–∞—Å—Ç–æ–ª—å–∫–æ –ø—Ä–æ—Å—Ç—ã, —á—Ç–æ –Ω–µ –±—É–¥—É—Ç –ø–æ–¥—Ä–æ–±–Ω–æ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—Ç—å—Å—è –ø–æ–∑–∂–µ. –§–∞–∫—Ç–∏—á–µ—Å–∫–∏, –Ω–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ –æ–Ω–∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è —Ä–µ–¥–∫–æ.

## –ü–æ–¥—Ä–æ–±–Ω—ã–µ –æ–±—ä—è—Å–Ω–µ–Ω–∏—è –æ–ø–µ—Ä–∞—Ü–∏–π —Å –∫–∞–Ω–∞–ª–∞–º–∏

–ß—Ç–æ–±—ã —Å–¥–µ–ª–∞—Ç—å –æ–±—ä—è—Å–Ω–µ–Ω–∏—è –æ–ø–µ—Ä–∞—Ü–∏–π —Å –∫–∞–Ω–∞–ª–∞–º–∏ –ø—Ä–æ—Å—Ç—ã–º–∏ –∏ –ø–æ–Ω—è—Ç–Ω—ã–º–∏, –≤ –æ—Å—Ç–∞–≤—à–µ–π—Å—è —á–∞—Å—Ç–∏ —Å—Ç–∞—Ç—å–∏ –∫–∞–Ω–∞–ª—ã –±—É–¥—É—Ç —Ä–∞–∑–¥–µ–ª–µ–Ω—ã –Ω–∞ —Ç—Ä–∏ –∫–∞—Ç–µ–≥–æ—Ä–∏–∏:

1. `nil`-–∫–∞–Ω–∞–ª—ã.
2. –ù–µ `nil`, –Ω–æ –∑–∞–∫—Ä—ã—Ç—ã–µ (closed) –∫–∞–Ω–∞–ª—ã.
3. –ù–µ –∑–∞–∫—Ä—ã—Ç—ã–µ –Ω–µ `nil`-–∫–∞–Ω–∞–ª—ã.

–í —Å–ª–µ–¥—É—é—â–µ–π —Ç–∞–±–ª–∏—Ü–µ –∫—Ä–∞—Ç–∫–æ —Å—É–º–º–∏—Ä–æ–≤–∞–Ω—ã –ø–æ–≤–µ–¥–µ–Ω–∏—è –≤—Å–µ—Ö –≤–∏–¥–æ–≤ –æ–ø–µ—Ä–∞—Ü–∏–π, –ø—Ä–∏–º–µ–Ω—è–µ–º—ã—Ö –∫ `nil`, –∑–∞–∫—Ä—ã—Ç—ã–º –∏ –Ω–µ –∑–∞–∫—Ä—ã—Ç—ã–º –Ω–µ `nil`-–∫–∞–Ω–∞–ª–∞–º.

| –û–ø–µ—Ä–∞—Ü–∏—è           | `nil`-–∫–∞–Ω–∞–ª   | –∑–∞–∫—Ä—ã—Ç—ã–π –∫–∞–Ω–∞–ª | –Ω–µ –∑–∞–∫—Ä—ã—Ç—ã–π –Ω–µ `nil`-–∫–∞–Ω–∞–ª |

–î–ª—è –ø—è—Ç–∏ —Å–ª—É—á–∞–µ–≤, –ø–æ–∫–∞–∑–∞–Ω–Ω—ã—Ö –±–µ–∑ –∏–Ω–¥–µ–∫—Å–æ–≤, –ø–æ–≤–µ–¥–µ–Ω–∏–µ –∫–∞–Ω–∞–ª–æ–≤ –æ—á–µ–≤–∏–¥–Ω–æ:

1. –ó–∞–∫—Ä—ã—Ç–∏–µ `nil`-–∫–∞–Ω–∞–ª–∞ –∏–ª–∏ —É–∂–µ –∑–∞–∫—Ä—ã—Ç–æ–≥–æ –∫–∞–Ω–∞–ª–∞ –≤—ã–∑—ã–≤–∞–µ—Ç –ø–∞–Ω–∏–∫—É –≤ —Ç–µ–∫—É—â–µ–π –≥–æ—Ä—É—Ç–∏–Ω–µ.
2. –û—Ç–ø—Ä–∞–≤–∫–∞ –∑–Ω–∞—á–µ–Ω–∏—è –≤ –∑–∞–∫—Ä—ã—Ç—ã–π –∫–∞–Ω–∞–ª —Ç–∞–∫–∂–µ –≤—ã–∑—ã–≤–∞–µ—Ç –ø–∞–Ω–∏–∫—É –≤ —Ç–µ–∫—É—â–µ–π –≥–æ—Ä—É—Ç–∏–Ω–µ.
3. –û—Ç–ø—Ä–∞–≤–∫–∞ –∑–Ω–∞—á–µ–Ω–∏—è –≤ `nil`-–∫–∞–Ω–∞–ª –∏–ª–∏ –ø–æ–ª—É—á–µ–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è –∏–∑ `nil`-–∫–∞–Ω–∞–ª–∞ –ø—Ä–∏–≤–æ–¥–∏—Ç –∫ –≤–µ—á–Ω–æ–π –±–ª–æ–∫–∏—Ä–æ–≤–∫–µ —Ç–µ–∫—É—â–µ–π –≥–æ—Ä—É—Ç–∏–Ω—ã.

–î–∞–ª–µ–µ –±—É–¥—É—Ç –¥–∞–Ω—ã –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–µ –æ–±—ä—è—Å–Ω–µ–Ω–∏—è –¥–ª—è —á–µ—Ç—ã—Ä–µ—Ö —Å–ª—É—á–∞–µ–≤, –ø–æ–º–µ—á–µ–Ω–Ω—ã—Ö –∏–Ω–¥–µ–∫—Å–∞–º–∏ (*A*, *B*, *C* –∏ *D*).

–ß—Ç–æ–±—ã –ª—É—á—à–µ –ø–æ–Ω—è—Ç—å —Ç–∏–ø—ã –∏ –∑–Ω–∞—á–µ–Ω–∏—è –∫–∞–Ω–∞–ª–æ–≤, –∞ —Ç–∞–∫–∂–µ —É–ø—Ä–æ—Å—Ç–∏—Ç—å –æ–±—ä—è—Å–Ω–µ–Ω–∏–µ –Ω–µ–∫–æ—Ç–æ—Ä—ã—Ö –∞—Å–ø–µ–∫—Ç–æ–≤ –∏—Ö —Ä–∞–±–æ—Ç—ã, –ø–æ–ª–µ–∑–Ω–æ —Ä–∞—Å—Å–º–æ—Ç—Ä–µ—Ç—å –∏—Ö –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–µ —Å—Ç—Ä—É–∫—Ç—É—Ä—ã.

–ú—ã –º–æ–∂–µ–º –ø—Ä–µ–¥—Å—Ç–∞–≤–∏—Ç—å –∫–∞–∂–¥—ã–π –∫–∞–Ω–∞–ª –∫–∞–∫ —Å–æ—Å—Ç–æ—è—â–∏–π –∏–∑ —Ç—Ä–µ—Ö –æ—á–µ—Ä–µ–¥–µ–π (–≤—Å–µ –æ–Ω–∏ –º–æ–≥—É—Ç –±—ã—Ç—å —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–Ω—ã –∫–∞–∫ FIFO-–æ—á–µ—Ä–µ–¥–∏) –Ω–∞ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–º —É—Ä–æ–≤–Ω–µ:

1. **receiving goroutine queue (RGQ)** –û—á–µ—Ä–µ–¥—å –≥–æ—Ä—É—Ç–∏–Ω –¥–ª—è –ø–æ–ª—É—á–µ–Ω–∏—è (–æ–±—ã—á–Ω–æ FIFO). –≠—Ç–∞ –æ—á–µ—Ä–µ–¥—å –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π —Å–≤—è–∑–∞–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫ –±–µ–∑ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è –ø–æ —Ä–∞–∑–º–µ—Ä—É. –ì–æ—Ä—É—Ç–∏–Ω–∫–∏ –≤ —ç—Ç–æ–π –æ—á–µ—Ä–µ–¥–∏ –Ω–∞—Ö–æ–¥—è—Ç—Å—è –≤ –∑–∞–±–ª–æ–∫–∏—Ä–æ–≤–∞–Ω–Ω–æ–º —Å–æ—Å—Ç–æ—è–Ω–∏–∏ –∏ –æ–∂–∏–¥–∞—é—Ç –ø–æ–ª—É—á–µ–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏–π –∏–∑ –∫–∞–Ω–∞–ª–∞.

2. **sending goroutine queue (SGQ)** –û—á–µ—Ä–µ–¥—å –≥–æ—Ä—É—Ç–∏–Ω –¥–ª—è –æ—Ç–ø—Ä–∞–≤–∫–∏ (–æ–±—ã—á–Ω–æ FIFO). –≠—Ç–∞ –æ—á–µ—Ä–µ–¥—å —Ç–∞–∫–∂–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π —Å–≤—è–∑–∞–Ω–Ω—ã–π —Å–ø–∏—Å–æ–∫ –±–µ–∑ –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è –ø–æ —Ä–∞–∑–º–µ—Ä—É. –ì–æ—Ä—É—Ç–∏–Ω–∫–∏ –≤ —ç—Ç–æ–π –æ—á–µ—Ä–µ–¥–∏ –Ω–∞—Ö–æ–¥—è—Ç—Å—è –≤ –∑–∞–±–ª–æ–∫–∏—Ä–æ–≤–∞–Ω–Ω–æ–º —Å–æ—Å—Ç–æ—è–Ω–∏–∏ –∏ –æ–∂–∏–¥–∞—é—Ç –æ—Ç–ø—Ä–∞–≤–∫–∏ –∑–Ω–∞—á–µ–Ω–∏–π –≤ –∫–∞–Ω–∞–ª. –ó–Ω–∞—á–µ–Ω–∏–µ (–∏–ª–∏ –∞–¥—Ä–µ—Å –∑–Ω–∞—á–µ–Ω–∏—è, –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–∞), –∫–æ—Ç–æ—Ä–æ–µ –∫–∞–∂–¥–∞—è –≥–æ—Ä—É—Ç–∏–Ω–∞ –ø—ã—Ç–∞–µ—Ç—Å—è –æ—Ç–ø—Ä–∞–≤–∏—Ç—å, —Ç–∞–∫–∂–µ —Ö—Ä–∞–Ω–∏—Ç—Å—è –≤ –æ—á–µ—Ä–µ–¥–∏ –≤–º–µ—Å—Ç–µ —Å —ç—Ç–æ–π –≥–æ—Ä—É—Ç–∏–Ω–æ–π.

–ö–∞–∂–¥—ã–π –∫–∞–Ω–∞–ª –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ —Å–æ–¥–µ—Ä–∂–∏—Ç –º—å—é—Ç–µ–∫—Å –¥–ª—è –ø—Ä–µ–¥–æ—Ç–≤—Ä–∞—â–µ–Ω–∏—è –≥–æ–Ω–æ–∫ –¥–∞–Ω–Ω—ã—Ö –ø—Ä–∏ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–∏ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –æ–ø–µ—Ä–∞—Ü–∏–π.

### –û–ø–µ—Ä–∞—Ü–∏—è —Å –∫–∞–Ω–∞–ª–æ–º ‚Äî —Å–ª—É—á–∞–π *A*

package main

import (
	"fmt"
	"time"
)

func main() {
	c := make(chan int) // an unbuffered channel
	go func(ch chan<- int, x int) {
		time.Sleep(time.Second)
		// <-ch    // fails to compile
		// Send the value and block until the result is received.
		ch <- x*x // 9 is sent
	}(c, 3)
	done := make(chan struct{})
	go func(ch <-chan int) {
		// Block until 9 is received.
		n := <-ch
		fmt.Println(n) // 9
		// ch <- 123   // fails to compile
		time.Sleep(time.Second)
		done <- struct{}{}
	}(c)
	// Block here until a value is received by
	// the channel "done".
	<-done
	fmt.Println("bye")
}
```

	x, ok = <-c
	fmt.Println(x, ok) // 5 true

	fmt.Println(x, ok) // 0 false

	// The send will also panic if the above
	// close call is removed.

}

		select {
		case c <- v:

		}
	}
	tryReceive := func() string {
		select {
		case v := <-c: return v
		default: return "-" // go here if c is empty
		}
	}
	trySend("Hello!") // succeed to send
	trySend("Hi!")    // succeed to send
	// Fail to send, but will not block.
	trySend("Bye!")
	// The following two lines will
	// both succeed to receive.
	fmt.Println(tryReceive()) // Hello!
	fmt.Println(tryReceive()) // Hi!
	// The following line fails to receive.
	fmt.Println(tryReceive()) // -
}
```
